Technical Stack

Under the hood

Kotlin + Jetpack Compose

100% Compose UI. MVVM architecture. Coroutines + StateFlow for reactive state management.

Room + SQLCipher

Schema v3. AES-256 encrypted database. Passphrase stored in Android Keystore via EncryptedSharedPreferences.

Foreground Service

MediaSession + MediaStyle notification. SCREEN_ON BroadcastReceiver for auto-relaunching lock screen Activity.

showWhenLocked API

Same tech as Google Maps navigation. Full Compose Activity rendered over the lock screen. No special permissions.
